\begin{reco}
\textbf{Utilisation :} ouvrir le fichier du template voulu, remplacer tous les champs en \textit{italique} par les données du projet, puis compiler avec \texttt{pdflatex [fichier].tex}. \\
\textbf{Hébergement :} les sources \texttt{.tex} sont versionnées sur SharePoint Online sous \texttt{PMO / Référentiel / Templates / [phase] /}. \\
\textbf{Maintenance :} toute amélioration suite à un REX est appliquée dans ce dossier et documentée en changelog dans ce fichier index.
\end{reco}
